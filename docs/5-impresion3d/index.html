5 - Impresión 3D

Para este encargo usamos el software impresión 3D Ultimaker Cura.

Cura es un software de laminado 3D, muy necesario para realizar una impresión 3D. Este tipo de impresiones requieren de un software de corte, comúnmente llamado laminador como Cura. Los programas de laminado permiten que el modelo 3D se convierta en un archivo que se encarga de darle las instrucciones a la impresora. Cura 3D es un software que toma el modelo en archivos STL u OBJ y procede a dividirlos en varias partes, dándoles forma de láminas. De esta manera las trabaja y luego las convierte en código máquina. Este lenguaje es Gcode el cual permite que la impresora entienda las instrucciones e imprima el modelo 3D.

Ventajas de usar Cura 3D Software

  • Tiene una interfaz intuitiva y fácil de utilizar la cual se puede configurar en español.
  • Es compatible con numerosas impresoras 3D y con su función Cura Connect se monitorean desde una misma interfaz.
  • Entre sus funciones permite modificar cambiando el tamaño del objeto, su posición, entre otras cosas.
  • Trabaja con archivos STL, OBJ y X3D pero además admite formatos de imágenes como JPG, IF, PNG, entre otros.
  • Realiza el laminado en un reducido tiempo.
  • Permite realizar configuraciones en la impresión según lo necesitado o utilizar el modo recomendado en las impresiones.
  • 3D Cura es compatible con un gran número de sistemas operativos.

Utilizando Cura!

LINK DE DESCARGA

https://ultimaker.com/es/software/ultimaker-cura
  • Paso 1: Setear Impresora

Añadir a una impresora que no este conectada a la red.

  • Paso 2: Escoger Impresora

Luego ir a Creality 3D > Creality Ender-3

  • Paso 3: Revisar Presets

Asegurar que el material sea de 1.75 mm

  • Paso 4: Escoger Material y Cabezal

En Material > eSun > PLA+ > Purple PLA+ El cabezal debe ser de 0.4 mm este corresponde al diametro por el cual sale el plastico de la impresora.

  • Paso 5: Abrir un Archivo STL

Si fue creado en Fusion 360 ir a Tools > Make, luego guardar el archivo.

  • Paso 6: Cambiar Presets

Cambiar Layer Height : 0.3 mm

  • Paso 7: SLICE

Luego apretar el botónm SLICE que se encuentra en la parte inferior derecha de la pantalla y hacer un preview.
